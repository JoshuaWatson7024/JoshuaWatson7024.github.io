Calculator App Reflection

  • In this project, I used typeScript, HTML, and CSS
  • With React, I build my page with JSX.Elements
  • I kept track of global variables with useState
  • My App uses two components, "ButtonTable" and "Display"
  • ButtonTable returns an HTML table of buttons. It takes multiple function, string, and boolean inputs
  • I styled the calculator with CSS
  • I increased the button and text size
  • I centered the calculator
  • I made it so unclickable buttons are red, while clickable buttons are green
  • I used a useState array to track the history of user input
  • History section styled as a sidebar using CSS
  • Clear history button resets all useState vars
  • I formatted the output to be readable (i.e. adding dashes when the value is cleared)
